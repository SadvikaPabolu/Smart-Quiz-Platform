## 📌 Overview
Smart Quiz Platform is a web-based application developed using the MERN stack. It provides an engaging environment where students can attempt quizzes, track their performance, and compete on leaderboards, while teachers can create and manage quizzes efficiently.

The platform is designed to make learning interactive, data-driven, and competitive.

## 🎯 Features
✔️ Role-based system (Student & Teacher)  
✔️ Create and manage quizzes  
✔️ Attempt quizzes with real-time evaluation  
✔️ Performance tracking and analytics  
✔️ Leaderboard ranking system  
✔️ Interactive and user-friendly UI  

---

## 👥 User Roles

### 🎓 Student
- Attempt quizzes  
- View scores and results  
- Track performance  
- Check leaderboard rankings  

### 👩‍🏫 Teacher
- Create quizzes  
- Manage questions  
- Monitor student performance  
- Analyze results

## 🧩 Modules Implemented

### 🔐 Authentication Module
- User registration & login  
- Role-based access control  
- Secure session handling  

---

### 📊 Student Dashboard
- Total score & quizzes completed  
- Average performance  
- Weekly progress tracking  
- Subject-wise analysis  

---

### 🧑‍🏫 Teacher Dashboard
- Manage quizzes (Create/Edit/Delete)  
- Track student participation  
- View performance analytics  

---

### 📝 Quiz Module
- Dynamic quiz creation  
- Multiple-choice questions  
- Timer and quiz settings  
- Interactive question navigation  

---

### 📈 Result & Evaluation
- Instant score calculation  
- Correct/incorrect answers  
- Accuracy percentage  
- Performance summary  

---

### 🏆 Leaderboard
- Ranking based on performance  
- Accuracy-based comparison  
- Time-based filters (Weekly / Monthly / All-time)  

---

## 🔄 Workflow
1. User registers/logs in  
2. Teacher creates quizzes  
3. Student attempts quizzes  
4. System evaluates responses  
5. Results and leaderboard are updated

## ⚡ Getting Started

### Prerequisites
- Node.js  
- MongoDB  

### Installation
```bash
# Clone the repository
git clone https://github.com/123lakshmi-majji/smart-quiz-platform.git

# Navigate to project folder
cd smart-quiz-platform

# Install dependencies
npm install

# Start the server
npm start
